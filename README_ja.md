## はじめに

Zditor は WYSIWYG の Markdown エディタです。標準 Markdown をサポートしつつ、構造化ドキュメント向けの拡張も備えています。現在は `Windows`、`macOS`、`Linux` に対応しています。

[Releases ページ](https://github.com/zditor/zditor-docs/releases) または [公式サイト](https://zditor.com) からダウンロードできます。

## スキル

`.claude/commands` では次の 2 つのスキルが利用できます。

- `zditor-syntax`: Zditor 拡張 Markdown 構文の学習と利用を支援します。
- `import-excel`: Excel 文書を Zditor 対応のデータベース表へ変換します。
