Zditor 内置了面向 PDF 的批注能力，既支持自由手写，也支持结构化的区域高亮批注。你可以在 PDF 上直接画线、框选、写批注、分图层管理，并把批注结果保存为独立的注释文件。

## 批注如何保存

PDF 批注不会直接写回原始 PDF 文件，而是保存在同目录、同文件名的独立 JSON 文件中：

```text
示例.pdf.zditor-pdf-annotation.json
```

这个文件会保存以下内容：

- 图层列表
- 手写笔触
- 区域高亮批注
- 批注颜色、宽度、评论、标题等信息

<Tip>
  这种 sidecar 文件方式的好处是不会破坏原 PDF，也便于同步、备份和版本管理。
</Tip>

## 界面结构

PDF 批注界面主要分为三部分：

### 顶部批注工具栏

用于切换标注工具、设置颜色、调整线宽、修改笔触参数，以及执行撤销和清空当前页操作。

### 次级工具栏

用于：

- 打开书签侧边栏
- 打开图层与批注侧边栏
- 打开搜索侧边栏
- 缩放页面
- 通过页码快速跳转

### 左侧面板

左侧面板根据当前标签页切换显示：

- PDF 书签
- 图层和批注列表
- 搜索结果

## 批注工具

### 指针工具

指针工具主要用于选中对象，而不是绘制。

在指针模式下，你可以：

- 选中已有笔触
- 选中已有区域高亮
- 查看高亮的快捷浮层
- 调整已存在区域高亮的尺寸

如果点击空白区域，会取消当前选择。

### 画笔

画笔用于自由手写，适合做手写圈画、勾线和任意轨迹标注。

可调参数包括：

- 颜色
- 线宽
- 粗细变化
- 平滑度
- 路径简化
- 起笔渐细
- 收笔渐细
- 起笔圆帽
- 收笔圆帽

<Tip>
  这些参数会影响笔锋、平滑程度和收尾形状，适合根据触控笔或鼠标习惯微调手感。
</Tip>

### 高亮笔

高亮笔也是自由手写，但与普通画笔不同：

- 默认半透明显示
- 更适合覆盖原文做重点标记
- 视觉效果更接近荧光笔

### 橡皮

橡皮工具会按路径擦除命中的笔触。

特点：

- 作用对象是手写笔触
- 擦除范围与线宽有关
- 更适合删除局部手写标记

<Warning>
  橡皮不会直接擦除区域高亮批注。区域高亮需要通过高亮工具栏、右键或侧边栏删除。
</Warning>

### 直线与矩形

如果你不想手动画得太随意，可以使用：

- **直线工具**：通过起点和终点绘制直线
- **矩形工具**：通过对角拖拽绘制矩形边框

适合做结构化标记、框重点内容或画辅助线。

### 套索

套索工具用于圈选已有笔触。

圈选后可以对选中的笔触执行：

- 删除
- 复制
- 剪切
- 粘贴
- 移动到其他图层
- 拖动整体平移位置

<Tip>
  套索只针对手写笔触，不是用于框选 PDF 原始文字。
</Tip>

### 激光笔

激光笔是临时视觉指示工具：

- 拖动时显示发光拖尾
- 停止后会自动消失
- 不会保存到批注数据中

这个工具更适合演示、讲解或共享屏幕时指示阅读位置。

### 手形平移

选择工具栏中的手形模式后，可以拖动页面进行平移，而不是进行批注。

适用于：

- PDF 放大后的查看
- 触控板或手写笔场景下快速拖动画布

## 区域高亮批注

区域高亮是 PDF 批注里最接近“结构化评论”的一种形式。

### 创建区域高亮

<Steps>
  <Step title="切换到区域高亮工具">
    选择工具栏中的区域高亮工具。
  </Step>

  <Step title="拖出一个矩形区域">
    在 PDF 页面上拖拽出一个高亮框。
  </Step>

  <Step title="填写标题和评论">
    松开鼠标后，会弹出一个编辑框，可以填写标题、评论，并选择颜色。
  </Step>

  <Step title="确认保存">
    点击确认按钮后，该区域高亮会被保存到当前图层。
  </Step>
</Steps>

创建时支持：

- 标题
- 评论
- 颜色
- 自动生成的区域缩略图

### 编辑区域高亮

创建完成后，区域高亮支持两种常见交互：

#### 在区域高亮工具下编辑

点击已有高亮区域后，会出现完整的编辑面板，可修改：

- 标题
- 评论
- 颜色

#### 在指针工具下查看

在指针模式下点击高亮，会出现一个轻量浮层，适合快速查看内容或进入编辑。

### 调整大小

当某个高亮处于编辑状态时，其边框周围会显示 8 个调整手柄。

你可以从以下方向拖拽调整：

- 左上、上、右上
- 左、右
- 左下、下、右下

这意味着高亮框支持非常细粒度的尺寸修正，而不是只能删除重画。

### 删除区域高亮

区域高亮可以通过以下方式删除：

- 在浮层中点击删除
- 在编辑工具栏中点击删除
- 在左侧批注列表中删除
- 在区域高亮工具下右键命中某个高亮区域删除

### 批注缩略图

每个区域高亮在保存时都会尝试截取对应 PDF 区域作为缩略图，并交给编辑器的图片上传逻辑处理。

缩略图的存储结果可能是：

- 本地绝对路径
- GitHub 图床 URL
- S3 图床 URL

如果某条批注缺失缩略图，可以在左侧批注列表中重新生成。

### 在 Markdown 中引用 PDF 卡片注释

PDF 区域高亮除了在 PDF 侧边栏中查看外，也可以作为卡片注释引用到 Markdown 文档中，用来把 PDF 阅读结果整理成可复用的笔记素材。

在 Markdown 文档中输入 `@` 后，会弹出一个引用列表。这个列表会展示当前工作区里的：

- Markdown 文档
- PDF 文档

当你在列表里选中某个 PDF 文档时，会继续展开二级菜单，里面通常包含两类内容：

- **PDF 文档本体**：用于引用整个 PDF 文件
- **该 PDF 下的高亮卡片**：每一条区域高亮批注都可以作为单独卡片被引用

如果你在二级菜单里选中某一条 PDF 高亮卡片，这条卡片就会被直接插入到当前 Markdown 文档中。

### 典型流程

<Steps>
  <Step title="先在 PDF 中做好区域高亮批注">
    给关键内容添加矩形高亮，并填写标题或评论，这样后续在引用列表里更容易识别。
  </Step>

  <Step title="回到 Markdown 文档输入 @">
    在编辑器里输入 `@`，打开工作区文档引用菜单。
  </Step>

  <Step title="选择目标 PDF">
    在一级列表中找到并选中目标 PDF 文档，打开它的二级菜单。
  </Step>

  <Step title="选择整份 PDF 或某条高亮卡片">
    如果想引用整份 PDF，就选 PDF 本体；如果想引用某个具体摘录，就选对应的高亮卡片。
  </Step>

  <Step title="插入到 Markdown 中">
    选中高亮卡片后，系统会把这条 PDF 卡片批注直接插入到当前 Markdown 文档。
  </Step>
</Steps>

### 点击 Markdown 中的 PDF Card

当 PDF 高亮卡片已经插入到 Markdown 文档后，点击这张 PDF Card，会弹出一个浮动工具栏。这个工具栏用于在“写作上下文”和“PDF 原文上下文”之间快速切换。

常见可操作元素包括：

- **跳转到 PDF**：直接打开并定位到对应的 PDF 文档
- **查看详情**：展开这条卡片批注的详细信息
- **查看缩略图**：查看该高亮区域对应的缩略图内容

这意味着 PDF Card 不只是一个静态引用块，而是一个可交互的批注节点。你可以在 Markdown 笔记里先看到结论，再随时回跳到 PDF 原文核对上下文。

<Tip>
  这种设计特别适合论文阅读和资料整理。Markdown 里保留的是“可写作的引用卡片”，而不是单纯的图片或文本摘录。
</Tip>

这个用法适合：

- 阅读论文时把关键段落摘录到笔记中
- 在 Markdown 文档里保留 PDF 原文缩略图和评论上下文
- 一边阅读 PDF，一边沉淀结构化研究笔记

<Tip>
  这种卡片式引用特别适合论文阅读、资料精读和知识整理场景。左边是 PDF 高亮区域，右边是 Markdown 文档中的引用效果，可以把“原文截图 + 批注说明”放在同一个写作流程里。
</Tip>

## 图层管理

图层是 PDF 批注管理的核心能力之一。

### 图层支持的操作

左侧图层面板支持：

- 新建图层
- 设为当前活动图层
- 双击图层名称直接重命名
- 点击隐藏按钮切换图层可见性
- 点击删除按钮移除图层

<Warning>
  删除图层时，该图层内的笔触也会一起被删除。
</Warning>

### 当前图层的意义

当前活动图层会影响：

- 新创建的手写笔触归属
- 新创建的区域高亮归属
- 撤销操作的作用范围
- 清空当前页时的作用范围

例如，“清空当前页”并不是删除整页所有内容，而是删除**当前页 + 当前图层**上的笔触。

### 图层显隐

图层可以单独隐藏或显示。隐藏后：

- 该图层的笔触不显示
- 该图层的区域高亮不显示
- 数据仍然保留，不会被删除

这很适合：

- 按主题拆分批注
- 区分个人笔记和团队批注
- 临时聚焦某一类标记

## 套索选择与批量操作

当你使用套索圈中若干笔触后，会出现一组垂直工具按钮。

可执行的操作包括：

- **删除**：删除当前选中的笔触
- **复制**：复制到 PDF 内部剪贴板
- **剪切**：剪切到内部剪贴板并删除原对象
- **粘贴**：粘贴到当前页，并自动做一点偏移避免完全重叠
- **移动图层**：把这些笔触整体移到另一个图层

此外，你还可以直接拖动这些被套索选中的笔触，整体移动位置。

### 删除快捷键

在指针模式或套索模式下，如果已有选中笔触，按以下键可以直接删除：

- `Delete`
- `Backspace`

## 撤销与清空

### 撤销

顶部工具栏的撤销按钮会撤销：

- 当前页
- 当前活动图层

中的最后一条笔触。

### 清空当前页

清空按钮会删除：

- 当前页
- 当前活动图层

上的所有笔触。

<Warning>
  当前实现里，“清空当前页”针对的是笔触，不是区域高亮批注。
</Warning>

## 书签、搜索与导航

### PDF 书签

如果 PDF 自身包含目录或书签结构，Zditor 会自动提取出来，显示在左侧书签面板中。

你可以：

- 展开和折叠多级书签
- 点击书签跳转到对应页
- 在侧边栏中高亮当前页对应的书签节点

### 搜索正文和批注

PDF 搜索不只会查正文，也会查区域高亮批注中的评论内容。

搜索支持两类结果：

- **正文文本结果**
- **批注评论结果**

搜索行为特点：

- `Cmd/Ctrl + F` 快速打开搜索
- 输入后会自动防抖搜索
- `Enter` 跳下一个结果
- `Shift + Enter` 跳上一个结果
- `Esc` 关闭搜索

### 搜索结果显示

搜索命中后：

- 页面上会高亮显示对应区域
- 左侧搜索面板会列出所有结果
- 如果命中的是批注评论，会显示批注颜色和评论摘要

## 缩放与阅读

PDF 查看器支持：

- 工具栏按钮缩放
- `Ctrl/Cmd + 滚轮` 缩放
- Safari / WKWebView 下的双指捏合缩放
- 点击页码直接输入跳转

### 深色模式

在深色主题下，PDF 页面会做反色显示，以提升阅读体验。

<Tip>
  反色只作用于底层 PDF 画面，顶层标注本身不会跟着反色，因此批注颜色仍然保持可读。
</Tip>

## 性能说明

PDF 页面列表采用了接近虚拟滚动的策略：

- 当前视口附近的页面会真实渲染
- 视口外页面会用占位块保持滚动高度

这样做的目的是在多页 PDF 场景下减少渲染压力，尤其是在高缩放和高分辨率屏幕上。

## 常见问题

<AccordionGroup>
  <Accordion title="批注数据保存在哪里？" icon="question">
    保存在原 PDF 同目录、同文件名的 `.zditor-pdf-annotation.json` 文件中，而不是直接写回 PDF。
  </Accordion>

  <Accordion title="为什么看不到某些批注？" icon="question">
    先检查该批注所在图层是否被隐藏。另外，当前活动图层只影响新建和部分操作范围，不代表其他图层数据不存在。
  </Accordion>

  <Accordion title="为什么搜索不到我写的批注？" icon="question">
    当前搜索的是区域高亮批注里的评论内容，不是所有手写笔迹的视觉内容。也就是说，纯手写线条本身不能被全文检索。
  </Accordion>

  <Accordion title="缩略图没有生成怎么办？" icon="question">
    可以在左侧批注列表中使用“重新生成缩略图”功能。缩略图生成依赖当前页面截图和图片上传配置。
  </Accordion>

  <Accordion title="删除图层会发生什么？" icon="question">
    删除图层会同时删除该图层中的笔触数据。请先确认是否需要备份或移动内容到其他图层。
  </Accordion>

  <Accordion title="区域高亮和普通高亮笔有什么区别？" icon="question">
    普通高亮笔是自由手写轨迹，更接近荧光笔涂抹；区域高亮是结构化矩形批注，支持标题、评论、缩略图、颜色和尺寸调整。
  </Accordion>
</AccordionGroup>
