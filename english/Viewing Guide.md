# Viewing Guide

This checklist selects a subset of the English movie directory. Each item keeps a checkbox state and points to a `ClassicMovieEnglish` record rendered as a SuperTag.

!!! note Status
    `- [x]` means watched. `- [ ]` means planned. This guide does not change the `watched` field inside each movie file.

## Checklist

- [x] Power and family: [The Godfather](./movies/The%20Godfather.md|mode=supertag)
- [x] Suspense and observation: [Rear Window](./movies/Rear%20Window.md|mode=supertag)
- [x] Trial room pressure: [12 Angry Men](./movies/12%20Angry%20Men.md|mode=supertag)
- [x] Space and metaphysics: [2001 A Space Odyssey](./movies/2001%20A%20Space%20Odyssey.md|mode=supertag)
- [x] Film noir entry point: [Double Indemnity](./movies/Double%20Indemnity.md|mode=supertag)
- [x] Silent-era essential: [City Lights](./movies/City%20Lights.md|mode=supertag)
- [ ] War and breakdown: [Apocalypse Now](./movies/Apocalypse%20Now.md|mode=supertag)
- [ ] Media and power: [Network](./movies/Network.md|mode=supertag)
- [ ] Late western gravity: [Unforgiven](./movies/Unforgiven.md|mode=supertag)
- [ ] Corruption in the city: [Chinatown](./movies/Chinatown.md|mode=supertag)
- [ ] Musical classic: [Singin' in the Rain](./movies/Singin'%20in%20the%20Rain.md|mode=supertag)
- [ ] Animation landmark: [Toy Story](./movies/Toy%20Story.md|mode=supertag)

## Three Good Next Picks

- If you want a war film first, start with [Apocalypse Now](./movies/Apocalypse%20Now.md|mode=supertag).
- If you want New Hollywood noir, start with [Chinatown](./movies/Chinatown.md|mode=supertag).
- If you want a lighter change of rhythm, start with [Singin' in the Rain](./movies/Singin'%20in%20the%20Rain.md|mode=supertag).
