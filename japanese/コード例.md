# コード例

このドキュメントでは、Zditor 文書でコード例を書く代表的な方法を示します。Inline Code、主要言語のコードブロック、設定ファイル、API リクエスト、diff 例を含みます。

## Inline Code

段落の中に `const app = "zditor"`、`npm run dev`、`docs.zditor.com`、`POST /api/documents` のような Inline Code を直接書けます。

Inline Code に向いている内容:

- コマンド: `git status`
- 変数名: `userId`
- 関数名: `renderMarkdown()`
- 設定キー: `enableCache`
- ファイルパス: `japanese/コード例.md`

## TSX / React

```tsx
import { useState } from "react";

type CounterProps = {
  initial?: number;
};

export function Counter({ initial = 0 }: CounterProps) {
  const [count, setCount] = useState(initial);

  return (
    <button onClick={() => setCount((value) => value + 1)}>
      Count: {count}
    </button>
  );
}
```

## Python

```py
def fibonacci(n: int) -> list[int]:
    values = [0, 1]
    while len(values) < n:
        values.append(values[-1] + values[-2])
    return values[:n]


print(fibonacci(8))
```

## SQL

```sql
SELECT
  author_id,
  COUNT(*) AS post_count,
  MAX(published_at) AS latest_post_at
FROM posts
WHERE status = 'published'
GROUP BY author_id
ORDER BY latest_post_at DESC;
```
