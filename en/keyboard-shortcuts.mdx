## Global shortcuts

### Search and save

| Shortcut | Action |
| --- | --- |
| `Mod-K` | Open the search box |
| `Mod-S` | Save the current document |

### View zoom

| Shortcut | Action |
| --- | --- |
| `Mod-=` | Zoom in |
| `Mod--` | Zoom out |
| `Mod-0` | Reset zoom |

## Editor operations

### Basic editing

| Shortcut | Action |
| --- | --- |
| `Mod-A` | Select the whole document |
| `Escape` | Blur the current editor focus |
| `Shift-Enter` | Insert a hard line break |

### Undo and redo

| Shortcut | Action |
| --- | --- |
| `Mod-Z` | Undo |
| `Mod-Y` | Redo |
| `Shift-Mod-Z` | Alternate redo shortcut |
| `Backspace` | Undo certain input rules |

## Code block editing

### Create code blocks

| Shortcut | Action |
| --- | --- |
| `Mod-Shift-\`` | Toggle a code block |

| Shortcut | Action |
| --- | --- |
| `ArrowUp` | Move up or leave the code block |
| `ArrowDown` | Move down or leave the code block |
| `ArrowLeft` | Move left or leave the code block |
| `ArrowRight` | Move right or leave the code block |

| Shortcut | Action |
| --- | --- |

| --- | --- |

| `Mod-Backspace` | Delete the math node and leave it |

| `Mod-Enter` | Exit math editing |

### Math navigation

| Shortcut | Action |
| --- | --- |

| `ArrowRight` | Move right or exit the math node |
| `ArrowUp` | Move up or exit the math node |

### Table navigation

| Shortcut | Action |
| --- | --- |

### Table editing

| Shortcut | Action |
| --- | --- |

| Shortcut | Action |
| --- | --- |

## Shortcut customization

<Tip>
  Some versions of Zditor allow custom key bindings in the settings panel. Core read-only shortcuts, such as global actions and some structural navigation keys, may remain fixed.
</Tip>

## FAQ

<AccordionGroup>
  <Accordion title="Why does a shortcut not work?" icon="question">
    Check whether the editor has focus, whether the current node type supports that action, and whether the operating system intercepted the shortcut first.
  </Accordion>

  <Accordion title="Are shortcuts different on macOS?" icon="question">
    The bindings are the same conceptually. Replace `Mod` with `Command` on macOS and `Ctrl` elsewhere.
  </Accordion>

  <Accordion title="Can I customize everything?" icon="question">
    Usually no. Some shortcuts are reserved by the editor or by the operating system.
  </Accordion>
</AccordionGroup>
