侧边文件树是 App 的核心功能之一，提供了直观、高效的文件和文件夹管理界面。本文档将帮助你快速掌握文件树的各种操作技巧。

## 界面概览

侧边文件树位于 App 的左侧，默认显示当前工作区的所有文件和文件夹。

![文件树布局](https://download.zditor.com/uploads/2025-12-30/1767098416_a892.png)

### 顶部工具栏

文件树顶部提供了以下功能按钮：

- **新建文件** - 在根目录创建新文件
- **新建文件夹** - 在根目录创建新文件夹
- **清空剪贴板** - 清除已复制/剪切的文件状态（仅在剪贴板有内容时显示）
- **过滤器** - 按文件类型过滤显示
- **更多操作** - 刷新、折叠所有、在资源管理器 (mac中是finder) 中打开等

![顶部工具栏](https://download.zditor.com/uploads/2025-12-30/1767098468_e055.png)

### 文件树列表

显示工作区中的所有文件和文件夹，支持缩进显示层级关系。

### 底部状态栏

显示当前工作区名称，点击 `可切换工作区` ，点击左侧按钮快速添加文件，点击右侧按钮弹出向上的菜单。

![底部状态栏](https://download.zditor.com/uploads/2025-12-30/1767098497_18f2.png)

## 基础操作

<AccordionGroup>
  <Accordion title="单击操作" icon="mouse">
    - **单击文件** - 在编辑器中打开文件
    - **单击文件夹** - 展开/折叠文件夹
  </Accordion>

  <Accordion title="右键菜单" icon="menu">
    - **文件右键** - 打开文件上下文菜单，支持复制、重命名等操作
    - **文件夹右键** - 打开文件夹上下文菜单，支持新建文件/文件夹、粘贴等操作
    - **底部空白区域右键** - 类似文件夹上下文菜单，支持新建文件/文件夹、粘贴等操作

    ![文件树操作演示](https://download.zditor.com/uploads/2025-12-30/1767099426_ce21.gif)
  </Accordion>
</AccordionGroup>

### 选择文件

单击文件或文件夹即可选中，选中项会高亮显示。

### 重命名

<AccordionGroup>
  <Accordion title="重命名方法" icon="pen">
    1. 右键点击文件或文件夹弹出菜单栏，选择"重命名"
    2. 或者在 hover 状态下点击更多按钮弹出菜单栏点击重命名
    3. 输入新名称后按 **Enter** 确认，按 **Esc** 取消
  </Accordion>
</AccordionGroup>

<Tip>
  **提示**：重命名文件时，扩展名会被自动选中，方便你只修改文件名主体部分。
</Tip>

## 文件与文件夹管理

### 新建文件/文件夹

<AccordionGroup>
  <Accordion icon="plus" title="方式一：使用顶部工具栏">
    点击工具栏的"新建文件"或"新建文件夹"按钮。
  </Accordion>

  <Accordion icon="folder-plus" title="方式二：使用快捷按钮">
    鼠标悬停在文件夹上时，会显示快捷操作按钮：

    ![悬停按钮组](https://download.zditor.com/uploads/2025-12-30/1767099518_9353.png)

    - **➕** - 在此文件夹中新建文件
    - **📁** - 在此文件夹中新建子文件夹
  </Accordion>

  <Accordion icon="mouse-right-click" title="方式三：右键菜单">
    右键点击文件夹，选择"新建文件"或"新建文件夹"。

    ![新建文件](https://download.zditor.com/uploads/2025-12-30/1767099547_5e5d.png)
  </Accordion>
</AccordionGroup>

### 删除文件/文件夹

1. 右键点击要删除的文件或文件夹
2. 选择"删除"
3. 确认删除操作

<Warning>
  **注意**：删除操作不可恢复，请谨慎操作。
</Warning>

### 复制/剪切/粘贴

<AccordionGroup>
  <Accordion icon="copy" title="复制文件">
    1. 右键点击文件，选择"复制"
    2. 文件会显示黄色的"复制"标记

    ![复制](https://download.zditor.com/uploads/2025-12-30/1767099594_c544.png)
  </Accordion>

  <Accordion icon="scissors" title="剪切文件">
    1. 右键点击文件，选择"剪切"
    2. 文件会显示青色的"剪切"标记

    ![剪切标记](https://download.zditor.com/uploads/2025-12-30/1767099613_b832.png)
  </Accordion>

  <Accordion icon="paste" title="粘贴文件">
    1. 右键点击目标文件夹（或根目录）
    2. 选择"粘贴"
    3. 文件会被复制/移动到目标位置
  </Accordion>
</AccordionGroup>

<Tip>
  **提示**：不能将文件粘贴到其自身或其子目录中。
</Tip>

## 排序与过滤

### 文件类型过滤

点击顶部工具栏的过滤器按钮（🔍），可以选择显示特定类型的文件：

![过滤文件类型](https://download.zditor.com/uploads/2025-12-30/1767099649_ac0e.png)

- **All** - 显示所有文件
- **Markdown** - 仅显示 Markdown 文件
- **Documents** - 显示文档类型文件
- **Assets** - 显示资源文件（图片、视频等）
- **Code** - 显示代码文件
- **Excalidraw** - 显示 Excalidraw 文件
- **Other** - 显示其他类型文件

可以同时选择多个过滤器，过滤器图标上会显示当前激活的过滤器数量。

### 排序选项

点击底部的最近文件夹菜单（⋯），打开排序选项：

![排序选项](https://download.zditor.com/uploads/2025-12-30/1767099743_db4d.png)

#### 排序方式

- **按名称排序** (Aa) - 按文件名字母顺序排列
- **按创建时间排序** (📅) - 按文件创建时间排列
- **按修改时间排序** (🕐) - 按文件最后修改时间排列

#### 排序方向

- **升序** (↑) - 从小到大或从早到晚
- **降序** (↓) - 从大到小或从晚到早

#### 文件夹分组

- **文件夹分组** (📁) - 将所有文件夹显示在文件之前

## 拖拽操作

文件树支持强大的拖拽功能，让文件管理更加直观。

<AccordionGroup>
  <Accordion icon="file-import" title="移动文件/文件夹">
    1. 按住要移动的文件或文件夹
    2. 拖动到目标文件夹
    3. 松开鼠标完成移动

    ![拖拽移动文件](https://download.zditor.com/uploads/2025-12-30/1767099808_3abe.png)
  </Accordion>

  <Accordion icon="upload" title="从外部拖入文件">
    直接将系统文件管理器中的文件拖入文件树，可以将其复制到当前工作区。

    ![拖拽外部文件](https://download.zditor.com/uploads/2025-12-30/1767099842_1cef.png)
  </Accordion>

  <Accordion icon="home" title="拖拽到根目录">
    将文件拖拽到文件树的空白区域（根目录），可以将其移动到工作区根目录。
  </Accordion>
</AccordionGroup>

<Tip>
  **提示**：拖拽时目标文件夹会高亮显示，悬停 600 毫秒后会自动展开文件夹。
</Tip>

## 右键菜单

文件和文件夹都提供了丰富的右键菜单功能。

### 文件右键菜单

![文件右键菜单](https://download.zditor.com/uploads/2025-12-30/1767099869_e82c.png)

| 选项 | 功能说明 |
| --- | --- |
| **打开** | 在左侧编辑器打开文件 |
| **右侧分屏打开** | 在右侧面板打开文件 |
| **在新窗口打开** | 创建新窗口打开文件 |
| **重命名** | 重命名文件 |
| **复制** | 复制文件到剪贴板 |
| **剪切** | 剪切文件到剪贴板 |
| **粘贴** | 粘贴剪贴板中的文件（仅当目标为文件夹时可用） |
| **在文件夹中打开** | 在系统文件管理器中打开文件所在位置 |
| **复制文件路径** | 复制文件的完整路径到剪贴板 |
| **删除** | 删除文件 |

### 文件夹右键菜单

![文件夹右键菜单](https://download.zditor.com/uploads/2025-12-30/1767100024_43c.png)

文件夹右键菜单包含文件的所有选项，以及：

| 选项 | 功能说明 |
| --- | --- |
| **新建文件** | 在此文件夹中创建新文件 |
| **新建文件夹** | 在此文件夹中创建子文件夹 |

### 根目录右键菜单

在文件树的空白区域右键点击，可以打开根目录菜单：

![根目录右键菜单](https://download.zditor.com/uploads/2025-12-30/1767100061_b76e.png)

包含所有文件操作以及：

- **刷新** - 重新扫描工作区文件
- **折叠所有** - 折叠所有展开的文件夹
- **在文件夹中打开** - 在系统文件管理器中打开工作区

## 工作区管理

### 切换工作区

1. 点击底部状态栏中的工作区名称
2. 在系统文件夹选择器中选择新的工作区
3. 文件树会自动加载新工作区的内容

### 最近文件夹

点击底部状态栏右侧的"更多"按钮（⋯），可以查看和访问最近使用的文件夹：

![最近文件夹](https://download.zditor.com/uploads/2025-12-30/1767100229_406f.png)

<AccordionGroup>
  <Accordion icon="thumbtack" title="固定文件夹">
    点击文件夹右侧的图钉图标（📌），可以将重要文件夹固定在列表顶部。
  </Accordion>

  <Accordion icon="trash" title="删除最近记录">
    对于未固定的文件夹，点击删除图标（🗑️）可以从最近列表中移除。
  </Accordion>
</AccordionGroup>

### 自动刷新

文件树会自动监控工作区文件系统的变化：

- 当文件被创建、删除或重命名时，文件树会自动更新
- 顶部会显示蓝色的加载进度条

## 常见问题

<AccordionGroup>
  <Accordion title="为什么某些文件在文件树中不显示？" icon="question">
    检查是否启用了文件类型过滤器，尝试将过滤器设置为"All"。
  </Accordion>

  <Accordion title="拖拽文件时没有反应？" icon="question">
    确保目标文件夹已展开，或者拖拽到文件夹图标上等待 600 毫秒让其自动展开。
  </Accordion>

  <Accordion title="如何快速找到最近修改的文件？" icon="question">
    使用排序功能，选择"按修改时间排序"并设置为降序，最近的文件会显示在顶部。
  </Accordion>

  <Accordion title="可以同时打开多个文件吗？" icon="question">
    可以，每次点击文件都会在编辑器中打开一个新的标签页。
  </Accordion>
</AccordionGroup>
