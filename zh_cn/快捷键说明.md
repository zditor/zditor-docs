# 快捷键说明

本文档列出了 Zditor 编辑器中的所有快捷键。

:::tip 提示
在快捷键表示中，`Mod` 指 `Ctrl`（Windows/Linux）或 `Cmd`（macOS）键。
:::

## 全局快捷键

| 快捷键          | 功能           | 说明                  |
| --------------- | -------------- | --------------------- |
| `Mod-k`         | 弹出搜索框     | 快速搜索文档内容      |
| `Mod-s`         | 保存文档       | 保存当前文档          |
| `Mod-Equal`     | 放大           | 视图缩放              |
| `Mod-Minus`     | 缩小           | 视图缩放              |
| `Mod-Digit0`    | 重置缩放       | 恢复默认缩放级别      |

![全局快捷键示例](./images/shortcuts/global-shortcuts.gif)

## 编辑器操作

### 文本格式

这些快捷键用于切换文本的格式标记：

| 快捷键         | 功能           | 说明                  |
| -------------- | -------------- | --------------------- |
| `Mod-b`        | 切换粗体       | 为选中文本添加粗体    |
| `Mod-i`        | 切换斜体       | 为选中文本添加斜体    |
| `Mod-d`        | 切换删除线     | 为选中文本添加删除线  |
| `Mod-h`        | 切换高亮       | 为选中文本添加高亮    |
| `Mod-Shift-c`  | 切换行内代码   | 为选中文本添加代码标记|
| `Mod-r`        | 切换修订标记   | 为选中文本添加修订标记|

![文本格式快捷键](./images/shortcuts/text-formatting.gif)

### 节点操作

这些快捷键用于插入和编辑各种节点类型：

| 快捷键         | 功能               | 说明                    |
| -------------- | ------------------ | ----------------------- |
| `Mod-Shift-p`  | 设置为段落         | 将当前节点转换为段落    |
| `Mod-Shift-7`  | 切换有序列表       | 插入或切换为有序列表    |
| `Mod-Shift-8`  | 切换无序列表       | 插入或切换为无序列表    |
| `Mod-Shift-9`  | 切换复选框列表     | 插入或切换为复选框列表  |
| `Mod-Shift-b`  | 切换引用块         | 插入或切换为引用块      |
| `Mod-Shift-h`  | 插入水平线         | 插入分隔线              |
| `Mod-p`        | 插入分页符         | 插入分页符              |
| `Mod-Shift-e`  | 插入行内公式       | 插入数学公式（行内）    |
| `Mod-Shift-d`  | 插入块级公式       | 插入数学公式（块级）    |
| `Mod-t`        | 创建3x3表格        | 插入3x3表格             |
| `Mod-Shift-\`  | 切换代码块         | 插入或切换为代码块      |
| `Mod-Shift-m`  | 创建元数据视图     | 插入元数据块            |
| `Mod-Shift-v`  | 切换HTML块         | 插入或切换为HTML块      |
| `Mod-a`        | 全选文档内容       | 选择整个文档            |
| `Escape`       | 失去焦点           | 退出编辑焦点            |

### 撤销重做

| 快捷键       | 功能                | 说明                  |
| ------------ | ------------------- | --------------------- |
| `Mod-z`      | 撤销                | 撤销上一步操作        |
| `Mod-y`      | 重做                | 重做操作（Mod+Y）     |
| `Shift-Mod-z`| 重做                | 重做操作（Shift+Mod+Z）|
| `Backspace`  | 撤销输入规则        | 撤销自动触发的输入规则|

![撤销重做](./images/shortcuts/undo-redo.gif)

## 编辑器列表

### 列表项操作

| 快捷键         | 功能               | 说明                    |
| -------------- | ------------------ | ----------------------- |
| `Enter`        | 分割列表项         | 在列表中创建新项        |
| `Tab`          | 缩进列表项         | 增加列表项缩进级别      |
| `Shift-Tab`    | 取消缩进列表项     | 减少列表项缩进级别      |
| `Mod-Shift-Enter` | 在列表中分割    | 在列表中强制换行        |
| `Alt-ArrowUp`  | 上移列表项         | 将当前项向上移动        |
| `Alt-ArrowDown`| 下移列表项         | 将当前项向下移动        |

## 编辑器代码

代码块的导航和编辑快捷键：

| 快捷键         | 功能                   | 说明                    |
| -------------- | ---------------------- | ----------------------- |
| `Backspace`    | 将代码块转换为段落     | 删除空的代码块          |
| `ArrowUp`      | 上移或退出代码块       | 在代码块中向上移动      |
| `ArrowLeft`    | 左移或退出代码块       | 在代码块中向左移动      |
| `ArrowDown`    | 下移或退出代码块       | 在代码块中向下移动      |
| `ArrowRight`   | 右移或退出代码块       | 在代码块中向右移动      |
| `Mod-Enter`    | 退出代码块             | 退出代码块编辑模式      |
| `Mod-Shift-f`  | 格式化代码块           | 格式化代码              |
| `Alt-Shift-f`  | 格式化代码块           | 格式化代码（备用快捷键）|

![代码块操作](./images/shortcuts/code-block.gif)

## 表格操作

表格内的导航和编辑快捷键：

| 快捷键               | 功能                     | 说明                    |
| -------------------- | ------------------------ | ----------------------- |
| `Tab`                | 移动到下一个单元格       | 在表格中向右移动        |
| `Shift-Tab`          | 移动到上一个单元格       | 在表格中向左移动        |
| `Mod-Enter`          | 添加行并移动             | 在表格中添加新行        |
| `Mod-Shift-Enter`    | 添加列并移动             | 在表格中添加新列        |
| `Mod-Backspace`      | 删除选中的行             | 删除当前行              |
| `Mod-Shift-Backspace`| 删除选中的列             | 删除当前列              |

![表格操作](./images/shortcuts/table.gif)
