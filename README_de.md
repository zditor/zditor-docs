## Einstieg

Zditor ist ein WYSIWYG-Markdown-Editor, der Standard-Markdown unterstuetzt und um strukturierte Dokumentfunktionen erweitert. Aktuell werden `Windows`, `macOS` und `Linux` unterstuetzt.

Downloads gibt es auf der [Releases-Seite](https://github.com/zditor/zditor-docs/releases) oder auf der [offiziellen Website](https://zditor.com).

## Skills

In `.claude/commands` stehen aktuell zwei Skills bereit:

- `zditor-syntax`: Hilft Agents beim Erlernen und Verwenden der erweiterten Zditor-Markdown-Syntax.
- `import-excel`: Konvertiert Excel-Dateien in von Zditor unterstuetzte Datenbanktabellen.
