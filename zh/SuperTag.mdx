SuperTag 是 Zditor 的结构化数据功能，它将 Markdown 文件组织为类似数据库的表格结构。通过在文档中插入 SuperTag，你可以引用特定表格中的行（文件），并以可视化标签的形式展示。

## 基本概念

在使用 SuperTag 之前，需要了解两个核心概念：

- **表格（Table）**：一组具有相同结构的文件集合，通过 Frontmatter 中的 `class` 字段定义
- **行（Row）**：属于某个表格的具体文件，其 Frontmatter 中的 `class` 值与表格名称匹配

## 创建 SuperTag

<AccordionGroup>
  <Accordion title="通过 # 菜单插入" icon="hashtag" defaultOpen>
    在编辑器中输入 `#`，会弹出 ObjectMenu 菜单：

    1. 输入 `#` 后显示所有可用的表格列表
    2. 选择一个表格，进入该表格的行列表
    3. 选择一行，即可插入 SuperTag 节点

    插入后的 SuperTag 显示为 `#表格名 · 文件名` 的彩色标签。
  </Accordion>

  <Accordion title="创建新表格" icon="plus">
    如果目标表格不存在，可以直接在菜单中创建：

    1. 输入 `#表格名`
    2. 在菜单底部点击「+ 添加 "表格名"」
    3. 系统会自动创建对应的表格定义文件
  </Accordion>

  <Accordion title="创建新行" icon="file-circle-plus">
    在表格的行列表中，可以直接添加新行：

    1. 选择表格后，在行列表底部点击「+ 添加行」
    2. 系统会创建一个新文件，自动设置 `class` 字段
    3. 新行会立即插入为 SuperTag
  </Accordion>
</AccordionGroup>

## Markdown 格式

SuperTag 在 Markdown 中以链接形式存储，通过 `mode=supertag` 参数标识：

```markdown
[表格名](path/to/file.md|mode=supertag)
```

编辑器会自动将其渲染为彩色标签 `#表格名 · 文件名`。

## 选中工具栏

点击选中一个 SuperTag 后，会弹出浮动工具栏，显示：

- **表格名** — 点击可跳转到表格视图
- **文件名** — 点击可打开对应文件
- **信息按钮** — 展开显示该文件的 Frontmatter 元数据字段，包括：
  - 复选框字段显示 ✓/✗
  - 进度字段显示进度条
  - 选择字段显示彩色标签
  - 日期、数字等字段以对应图标展示

## 颜色系统

SuperTag 根据表格名称自动分配颜色，共 8 种配色方案：

| 编号 | 颜色 |
|------|------|
| 0 | Indigo（靛蓝） |
| 1 | Rose（玫红） |
| 2 | Emerald（翠绿） |
| 3 | Amber（琥珀） |
| 4 | Cyan（青色） |
| 5 | Violet（紫罗兰） |
| 6 | Pink（粉色） |
| 7 | Teal（青绿） |

同一表格名称的 SuperTag 始终使用相同颜色，便于视觉区分。

## 与 Frontmatter 的关系

SuperTag 的表格系统依赖 Frontmatter 中的特殊字段：

**定义表格（Schema 文件）**：
```yaml
---
class:
  type: text
  value: "项目"
field:
  type: array
  value: [名称, 状态, 优先级]
---
```

**创建行（数据文件）**：
```yaml
---
class:
  type: text
  value: "项目"
名称:
  type: text
  value: "我的项目"
状态:
  type: select
  value: "进行中"
---
```

所有 `class` 值为 "项目" 的文件会自动归入同一表格，可在表格视图中统一查看和管理。

## 常见问题

<AccordionGroup>
  <Accordion title="SuperTag 的颜色可以自定义吗？">
    目前颜色由表格名称自动分配，无法手动指定。如果需要不同颜色，可以调整表格名称。
  </Accordion>

  <Accordion title="如何查看某个表格的所有行？">
    点击 SuperTag 浮动工具栏中的表格名称，即可跳转到该表格的数据视图，查看所有行。
  </Accordion>

  <Accordion title="删除 SuperTag 会删除对应文件吗？">
    不会。删除 SuperTag 只是移除文档中的引用标签，对应的文件不受影响。
  </Accordion>
</AccordionGroup>
