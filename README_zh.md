## 入门指南

Zditor是一个所见即所得的markdown编辑器，支持markdown基础语法，并在此基础上扩展。目前支持`Windows`、`Macos`、`Linux`平台，下载请前往 [release下载界面](https://github.com/zditor/zditor-docs/releases)。

### 签名问题

- `macos` 目前没有签名，下载之后需要额外两步处理
