Zditor 支持标准 Markdown 脚注语法，让你可以在文档中添加注释、引用来源或补充说明，而不打断正文的阅读流畅性。脚注引用显示在正文中，完整内容则集中展示在文档底部。

  </Card>

## 创建脚注

<AccordionGroup>
  <Accordion title="Markdown 语法" icon="code" defaultOpen>
    在正文中输入脚注引用标记 `[^数字]`：

    ```markdown
    这是一段包含脚注的文本[^1]，可以添加多个脚注[^2]。
    ```

    输入 `[^1]` 后，编辑器会自动将其转换为上标形式的脚注引用 `[1]`。
  </Accordion>

  <Accordion title="自动转换规则" icon="sparkles">
    **触发条件**：

    输入 `[^` + 数字 + `]` 后自动转换为脚注节点。

    **示例**：
    ```
    输入：这是正文[^1]
    结果：这是正文 [1]（上标显示）
    ```

    **注意**：
    - 标签仅支持数字（如 `[^1]`、`[^2]`）
    - 相同数字标签的脚注共享同一内容
  </Accordion>
</AccordionGroup>

## 编辑脚注内容

脚注创建后默认内容为空，你可以通过两种方式编辑脚注内容。

### 通过浮动工具栏编辑

<Steps>
  <Step title="选中脚注引用">
    点击正文中的脚注引用（如 `[1]`），选中该脚注节点。
  </Step>
  <Step title="查看浮动工具栏">
    选中后会弹出浮动工具栏，显示脚注标签和当前内容。如果尚未填写内容，会显示为空。
  </Step>
  <Step title="点击编辑按钮">
    点击工具栏中的编辑图标，进入编辑模式，出现输入框。
  </Step>
  <Step title="输入内容并保存">
    在输入框中填写脚注内容，按 `Enter` 保存，按 `Esc` 取消编辑。
  </Step>
</Steps>

### 通过文档底部编辑

当文档中存在脚注时，文档底部会自动出现脚注列表区域。

<Steps>
  <Step title="滚动到文档底部">
    在文档末尾的分隔线下方，可以看到所有脚注的汇总列表。
  </Step>
  <Step title="点击脚注内容">
    点击某条脚注的内容文本，进入行内编辑模式。
  </Step>
  <Step title="编辑并保存">
    修改内容后，按 `Enter` 或点击其他区域即可保存。
  </Step>
</Steps>

<Tip>
  编辑任意一处脚注内容后，文档中所有使用相同标签的脚注引用会自动同步更新。例如，如果你在多处使用了 `[^1]`，编辑一次内容即可全部更新。
</Tip>

## 脚注的显示

### 正文中的引用

脚注在正文中以上标形式显示，格式为 `[数字]`。鼠标悬停在脚注引用上时，会通过 tooltip 显示脚注内容。

### 文档底部列表

所有脚注会在文档底部自动汇总为有序列表，以分隔线与正文区分：

- 每条脚注显示标签编号和对应内容
- 相同标签的脚注会自动去重，只显示一次
- 列表按标签顺序排列

## 使用示例

```markdown
量子计算利用量子力学原理进行信息处理[^1]，
与经典计算机相比具有指数级的速度优势[^2]。

目前主流的量子计算方案包括超导量子比特[^1]和离子阱技术。
```

上面的示例中：
- `[^1]` 出现了两次，它们共享同一脚注内容
- `[^2]` 是独立的脚注
- 文档底部会显示两条脚注（`1` 和 `2`）

## 常见问题

<AccordionGroup>
  <Accordion title="脚注标签可以使用文字吗？">
    目前仅支持数字作为脚注标签（如 `[^1]`、`[^2]`）。不支持文字标签如 `[^note]`。
  </Accordion>

  <Accordion title="如何删除脚注？">
    在正文中选中脚注引用节点后，按 `Delete` 或 `Backspace` 键即可删除。删除所有相同标签的引用后，底部对应的脚注条目也会自动消失。
  </Accordion>

  <Accordion title="多个相同标签的脚注内容不一致怎么办？">
    编辑器会自动同步相同标签的脚注内容。当你编辑其中任意一个的内容时，所有相同标签的脚注都会更新为相同内容。
  </Accordion>

  <Accordion title="脚注在只读模式下如何显示？">
    在只读模式下，脚注引用和底部列表正常显示，但无法进行编辑操作。
  </Accordion>
</AccordionGroup>
