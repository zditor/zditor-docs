# Code Examples

这个目录包含可直接在 Zditor 中打开的代码示例文档，用来演示 Markdown 文档中如何组织和展示代码块。

当前示例覆盖了多种常见场景，包括：

- 基础语言：JavaScript、TypeScript、Python、Bash
- 前端代码：TSX / React、HTML、CSS
- 工程配置：JSON、YAML、Dockerfile
- 服务与数据：HTTP、SQL
- 其他语言：Go、Rust、Diff
- 大体量示例：上百行 TypeScript、上百行 Python
- 长行示例：超长 JSON、SQL、Bash、JSX 单行

- 主文档：[code-examples.md](code-examples.md)
- 适用场景：教程、API 文档、脚本说明、开发笔记
