# 快捷键说明

本文档列出了 Zditor 编辑器中一部分快捷键

## 全局快捷键

| 快捷键  | 功能       | 说明                       |
| ------- | ---------- | -------------------------- |
| `Mod-k` | 弹出搜索框 |                            |
| `Mod-s` | 保存       | 保存当前文档（不完成编辑） |

## 节点快捷键

编辑文档且选区为空时触发快捷键操作

| 快捷键        | 归属       | 功能               | 说明                                               |
| ------------- | ---------- | ------------------ | -------------------------------------------------- |
| `Mod-Shift-d` | Math       | 插入块公式         | d = Display（显示模式）→ 块级公式                  |
| `Mod-Shift-e` | Math       | 插入行内公式       | e = Equation / Expression（方程/表达式）→ 行内公式 |
| `Mod-Shift-m` | Metadata   | 插入元数据         | m = metadata → 行内公式                            |
| `Mod-Shift-c` | CodeInline | 切换/插入 内联代码 | c = CodeInline → 行内公式                          |
| `Mod-Shift-]` | BlockQuote | 切换/插入 引用     |                                                    |
| `Mod-]`       | BlockQuote | 切换/插入 引用     |                                                    |
| `Mod-Shift-b` | BlockQuote | 切换/插入 引用     | b = BlockQuote → 引用                              |
| `Shift-Enter` | BlockQuote | 换行               | 在Blockquote中使用，连续换行避免跳出blockquote     |
| `Mod-Shift-l` | List       | 切换/插入 列表     | l = List → 列表                                    |

## 标记快捷键

编辑文档且选中文本时触发快捷键操作

| 快捷键  | 归属          | 功能                 | 说明                      |
| ------- | ------------- | -------------------- | ------------------------- |
| `Mod-b` | Bold          | 切换当前选中文档标记 | b = Bold → 加粗           |
| `Mod-B` | Bold          | 切换当前选中文档标记 | B = Bold → 加粗           |
| `Mod-c` | CodeInline    | 切换/插入 内联代码   | c = CodeInline → 代码内联 |
| `Mod-C` | CodeInline    | 切换/插入 内联代码   | C = CodeInline → 代码内联 |
| `Mod-d` | Strikethrough | 切换/插入 删除波浪   | d = Delete → 删除         |
| `Mod-D` | Strikethrough | 切换/插入 删除波浪   | d = Delete → 删除         |
| `Mod-h` | Highlight     | 切换/插入 高亮       | h = Highlight → 高亮      |
| `Mod-H` | Highlight     | 切换/插入 高亮       | H = Highlight → 高亮      |
| `Mod-i` | Italic        | 切换/插入 斜体       | i = Italic → 斜体         |
| `Mod-I` | Italic        | 切换/插入 斜体       | I = Italic → 斜体         |
| `Mod-r` | Revision      | 切换/插入 修订       | r = Revision → 修订       |
| `Mod-R` | Revision      | 切换/插入 修订       | r = Revision → 修订       |
